# DISCUSSION.md

## Changes

- The app fetches and displays advocate data from a live PostgreSQL db using Drizzle ORM.
- Search functionality is working with support for multiple fields.
- UI was refactored using TailwindCSS for a clean and responsive layout.
- Hydration and TypeScript errors were resolved.
- Data seeding is functional and tested.
